A function must pass the horizontal line test, the line must touch only one point of the graph. Let's see how the graph f(x)=x^2+5 looks like.

The line touched two points of the graph. This means f(x)=x^2+5 is not a function. Consequently, the inverse is not a function. Now, nothing is a function.

In conclusion, those graphs that do not pass the horizontal test are not considered functions. By instance, no function inverses.
