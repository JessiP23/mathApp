System of Equations are a pairs of equations that are being used to solve.There are several ways to calculate answers in a system of equations. I will show an example to demonstrate the mechanism of working with situations like this.

Example: Solve the following system of equations: 4x + y = 4, and 2x - y = 8

To solve this problem, the main problem is to get rid of one of the variables. Let's say we want to get rid of y. In this example we add both equations.

Once having one value we substitute x in either one of the equations. I will use the first equation

Now we have found the solution (2,-4). The solution is mostly the intersection of those two equations.
