Rational Functions

Vertical and Horizontal Asymptotes

There are certain rules to find the Horizontal and Vertical Asymptotes. We'll be through most of the definitions while solving examples.

Example: Find the Vertical and Horizontal Asymptotes of f(x)=(3+2x^2)/(x^2+x-6)

Find the Vertical Asymptote

  • Factor denominator. Something IMPORTANT: In some ocassions, when numerator is factored there is the case that the same number makes numerator and denominator 0. If that were the case, just ignore that point and keep going. If when factor a number only makes the denominator 0, that number is then a Vertical asymptote.

Find the Horizontal Asymptote

  • The Horizontal Asymptote is based on the x exponents. If the highest exponent in the numerator = highest exponent in the denominator then we divide both highest terms and the coefficient resultant will be our Horizontal Asymptote

Knowing Horizontal Asymptote and Vertical Asymptote, we can predict how our function will behave.
